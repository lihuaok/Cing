# 周易智慧 - 决策辅助系统

> 弘扬中华优秀传统文化，提供专业的周易占卜与决策辅助功能

## 项目简介

《周易智慧》是一款基于传统周易文化开发的决策辅助系统，旨在通过现代技术手段传承和弘扬中华优秀传统文化。系统融合了《周易》六十四卦、阴阳学说、五行理论等核心思想，为用户提供专业的占卜服务和人生决策参考。

## 功能特性

### 核心功能

| 模块       | 功能描述                  |
| -------- | --------------------- |
| **卜卦模拟** | 传统金钱卦法，三枚铜钱摇卦模拟       |
| **周易基础** | 阴阳学说、八卦体系、六十四卦、五行学说详解 |
| **事业前程** | 基于卦象分析事业发展方向和机遇       |
| **财富格局** | 财运走向分析和理财建议           |
| **正缘桃花** | 姻缘运势和情感发展预测           |
| **健康五行** | 根据五行学说分析健康状况          |
| **流年运势** | 年度整体运势测算              |
| **一生命运** | 人生轨迹和发展趋势分析           |
| **经典研读** | 历代圣贤易学见解与现代应用         |

### 特色功能

- ✨ **易传箴言轮播**：经典周易名言自动轮播展示
- ✨ **生活智慧卡片**：周易思想的现代启示
- ✨ **名家论易专栏**：孔子、荀子、王弼等历代圣贤的易学见解
- ✨ **六十四卦指南**：完整的卦象解读体系
- ✨ **本地数据存储**：用户占卜记录安全存储于 localStorage

## 技术栈

```
├── 前端框架：原生 HTML5 + CSS3 + JavaScript (ES6+)
├── 构建工具：Webpack 5
├── 代码保护：TerserPlugin + 代码混淆
├── 字体资源：Google Fonts (Noto Serif SC, ZCOOL XiaoWei)
└── 数据存储：localStorage
```

## 快速开始

### 环境要求

- Node.js >= 14.0.0
- npm >= 6.0.0

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

访问 `http://localhost:8081` 查看效果

### 生产构建

```bash
npm run build
```

构建产物将输出到 `dist/` 目录

## 设计理念

### 视觉设计

- **主色调**：深棕色系（#3d2914）体现传统文化厚重感
- **点缀色**：玫红色（#c0506a）作为强调色，高雅内敛
- **金色系**：金棕色（#c9a227）增添尊贵感
- **书法字体**：ZCOOL XiaoWei 用于标题，Noto Serif SC 用于正文

### 交互体验

- 平滑页面切换动画
- 悬停反馈效果
- 渐变过渡动画
- 响应式布局适配
